# Little Lemon API project

## Steps taken

1. Create project directory
```
mkdir LittleLemon
cd LittleLemon
```

2. Setup Django
```
pipenv install django
```

3. Start virtual environment
```
pipenv shell
```

4. Create project
```
django-admin startproject LittleLemon .
```
